Task 1 bundle (script + STL)

1) Install:
   python -m pip install -r requirements.txt

2) Run:
   python task1_feb25.py

Keys:
  1/2/3 : Load cases A/B/C
  P     : Enter phi theta psi (deg)
  V     : Enter u v w (m/s) in BODY
  W     : Enter wind N E D (m/s)
  T     : Print state to terminal
  R     : Reset angles
  Q     : Quit
