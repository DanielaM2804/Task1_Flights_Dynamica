# Task 1 – Coordinate Frames and Transformation Matrix (Dinámica de Vuelo)

Aplicación interactiva en **Python + PyVista/VTK** para el **Trabajo 1** del curso de Dinámica de Vuelo.  
Permite transformar un vector de velocidad desde el **Body frame** hacia el frame **NED (Vehicle)** usando una **DCM**, calcular los ángulos **α, β, γ**, y visualizar el avión (STL) con una interfaz tipo HUD.
